Acerca de MiCine Web 🎬

MiCine Web es una plataforma ficticia creada para explorar el desarrollo web utilizando tecnologías como HTML, CSS y JavaScript puro (sin frameworks).

¿Qué puedes hacer aquí?

  • Explorar un catálogo de películas.
  • Agregar películas a tu carrito.
  • Comprar películas para guardarlas en tu biblioteca personal.

Objetivo del proyecto

Este proyecto fue desarrollado con fines educativos. Su propósito es poner en práctica habilidades relacionadas con:

  • Manipulación del DOM
  • Uso del localStorage
  • Diseño responsivo
  • Organización de un sitio web completo
